Leap year checker

In this project you will learn how to check wether the given year is a leap year or not

Project description

This projects teaches you how to check wether a given year is a leap year or not in Gregorian Calendar by using Javascript ternary operators
This project covers:

  1. Javascript window.prompt()prompt box
  2. Javascript ternary operator
  3. Javascript alert() functions

What are the requirements?

  • HTML basic
  • Javascript basic

1.1 Get the year and evaluate it using javascript ternary operators project1006.js 

//window.prompt() displays a dialog with an optional //message prompting the user to input message.
//It has two parameters:
//window,prompt(message,default)
//message-is a string of text to display to user;
//This is optional,you can omit it if you don't want to put any message in prompt box.
//default-is a string containing the default value displayed in the text input field.This is also optional.
//In this code we collected user input information by
//window.prompt() and assign it to variable named-year
 year=window.prompt("please input a year:");

//The condition for being leap year:
//if a year can be evenly devisible by 100 and 400,it can be leap year
//if a year not evenly devisible by 100 and 400 but if it can be evenly devisble by 4 then it also can be leap year. so,
x=(year % 100===0)?(year % 400===0):(year % 4===0);


//When one of these condition is true 'x' will have the value of 'true', then:
if(x===true)
{
    alert("This is a leap year.");
}

//When none of these condition is true 'x' will have the value of 'false', then:
else
{
    alert("This is not a leap year.");
}
