Print current page

In this project you will learn how to print current page

Project description

This projects teaches you how to work with javascript window.print()
This project covers:

  1. Javascript window.print() functions
  2. HTML button
  3. HTML onclick()attributes

What are the requirements?

  • HTML basic
  • Javascript basic

So let's start learning ...

Create the basic html code:

1.0 Create the basic html code with button project1002.html 

<!DOCTYPE HTML>
<html>
    <head>
        <meta charset="utf-8">
        <title>Print the current page</title>

    </head>

    <body>

        <h1 class="first-title">
            Click the button to print the current page
        </h1>

<!--call the window.print() function by pressing this button-->
<!--The window.print() function opens the print dialogue to print the current document-->

        <button  onclick="window.print()">
            Print this page
        </button>

        <!--This is the end-->

    </body>
</html>
