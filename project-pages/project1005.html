Rotate string from left to right


In this project you will learn how to how to rotate string from left to right using Javascript animation

Project description

This projects teaches you how to rotate the string from left to right using Javascript animation
This project covers:

  1. HTML onload attribute
  2. Javascript setInterval()functions
  3. textNode and childNode concept
  4. Javascript functions
  5. Javascript variables

What are the requirements?

  • HTML basic
  • Javascript basic

1.1 Create Javascript function to animate string project1005.js 

//Create a function 'animate-name()', and the parameter is the 'id' of the element we want to animate
function animate_name(id)
{
    //get the element we want to animate by ID and store it
    //in a variable named:element.
    var element = document.getElementById(id);

    //Just assume no other child exist
    //'element.childNodes[0]' will return the first child
    //so in this code we will get the first child and the only child of 'element'
    //and store it in a variable 'textNode'
    var textNode = element.childNodes[0];

    //'textNode.Data' will return all text content in
    //textNode and store it in a variable named:text
    var text = textNode.data;

    //This function will calls a function or execute an expression repeatedly as a spesified intervals
    //so it has two parameters:functions or expressions and time in 'milllisecond'
setInterval(
    //First parameter-function
    function ()
{

    //text[text.length-1] is the last character of a string:text
    //text.substring(0,text.length-1) are the all characters except the last characters
    //This line of code take one  letter from the last and attach it to the front
 text = text[text.length - 1] + text.substring(0,   text.length - 1);


    //assign the newly created string-'text' to textNode.data(all content of textNode)
  textNode.data = text;
    },

    //Second parameter-time
    1000);
}
