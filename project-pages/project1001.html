Display the currrent Day and Time

In this project you will learn how to display current Day and Time in a spesific format.

Project description

This projects teaches you how to work with javascript Date objects.
This project covers:

  1. Javascript Date objects
  2. Javascript getDay()functions
  3. Javascript getHours()functions
  4. Javascript getMinutes()functions
  5. Javascript getSeconds()functions
  6. Javascript window.alert() functions
  7. Javascript if, else statement
  8. Javascript variables

What are the requirements?

  • HTML basic
  • CSS basic
  • Javascript basic

1.1 Get the day and time from Date objects project1001.js 

//create new Date object and store it in a variable named:today
var today=new Date();

//get the day value of the new Date object by calling
//getDay() function and store it in a variable named:day
//this getDay function returns integers from 0-6.
//0 refer to sunday
//1 refer to monday
//2 refer to tuesday
//...
//6 refer to saturday
var day=today.getDay();

//create an array and store all the days of the week
//The index number of javascript array starts from 0.
//day[0] will be the first element
var days=["Sunday","Monday","Tuesday","Wednesday","Thursday","Friday","Saturday"];


//display the current day in the alert box
//by using window.alert() function.
//the integer value of day will be the array index for days and it will output the corresponding day value as a string
window.alert("The current day is "+days[day]+".");

//get the hour,minute and second value of the new Date object by calling getHour(),getMinute(),getSecond() function for this object
//and store them in a variables named:hour,minute and second
var hour=today.getHours();
var minute=today.getMinutes();
var second=today.getSeconds();

//This code help you decide AM or PM
//if the hour greater than or equal 12:
//it is PM,
//if it is less than 12:
//it is AM
var pam=(hour>=12)?"PM":"AM";

//This code help you make the range :00-12
//if the hour greater than or equal 12:
//it is hour-12,
//if it is less than 12:
//it is hour itself
hour=(hour>=12)? hour-12:hour;

//This code help you decide if it is noon or not
//first condition for being noon
if(hour===0 && pam==='PM')
{
    //second condition for being noon
    if(minute===0 && second===0)
    {
        hour=12;
        pam='noon';
    }
    else{
        hour=12;
        pam='PM';
    }
}

//This code help you decide if it is midnight or not
//first condition for being midnight
if(hour===0 && pam==='AM')
{
    //second condition for being midnight
    if(minute===0 && second===0)
    {
        hour=12;
        pam='midnight';
    }
    else{
        hour=12;
        pam='AM';
    }
}

//this code will output the current time
//in the alert box with a following style
window.alert("Now the time is "+ hour+" : "+minute+" : "+second + " "+pam);

//we can also output day and time in one alert box
window.alert("Today is "+days[day]+" and now the time is "+ hour+" : "+minute+" : "+second + " "+pam);
